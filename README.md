Bible Verse Generator by Feeling

A simple Python script that provides random Bible verses and short descriptions based on how the user is feeling. Perfect for encouragement, reflection, or daily inspiration.

Features

Supports multiple feelings: anxious, lonely, grateful, broken, tempted, hopeless, joyful, fearful.

Provides a random Bible verse and a short, feeling-based description.

Recognizes synonyms (e.g., "worried" → anxious, "sad" → broken).

Easy to extend with more verses or feelings.

Optional interactive mode for command-line use.
